# 🎨 Xalora Theme Preview

5 completely different light theme designs for Xalora. Choose one before we refactor the entire codebase!

## Files Structure

```
theme-preview/
├── index.html                      # Main preview page with all themes
├── 1-grid-indigo.html              # Theme 1: Grid Pattern + Indigo
├── 2-gradient-noise-blue.html      # Theme 2: Gradient + Noise + Blue
├── 3-glassmorphism-purple.html     # Theme 3: Glassmorphism + Purple
├── 4-warm-orange.html              # Theme 4: Warm Orange Gradient
├── 5-minimal-teal.html             # Theme 5: Minimal Teal Dot Grid
└── README.md                       # This file
```

## How to View

### Option 1: Open in Browser
1. Open `index.html` in your browser
2. Click on any theme card to view the preview
3. Each theme shows a complete home page design (navbar, hero, features, stats, footer)

### Option 2: Live Server
```bash
# If you have Python 3
python -m http.server 8000

# If you have Node.js http-server
npx http-server

# Then open: http://localhost:8000/theme-preview/
```

## Theme Overview

### 1️⃣ **Grid Indigo** (`1-grid-indigo.html`)
- **Background**: Soft gray-blue with subtle grid pattern (24px grid)
- **Accent Color**: Indigo (#4f46e5 → #7c3aed)
- **Style**: Modern, professional, corporate
- **Best For**: Enterprise clients, premium products
- **Vibe**: Clean, structured, trustworthy

### 2️⃣ **Gradient + Noise** (`2-gradient-noise-blue.html`)
- **Background**: Soft blue-purple gradient with noise texture
- **Accent Color**: Blue (#3b82f6 → #1d4ed8)
- **Style**: Premium, sophisticated, textured
- **Best For**: Premium learning platforms, high-end products
- **Vibe**: Elegant, premium, refined

### 3️⃣ **Glassmorphism** (`3-glassmorphism-purple.html`)
- **Background**: Light purple gradient with frosted glass cards
- **Accent Color**: Purple (#a855f7 → #d946ef)
- **Style**: Modern, trendy, contemporary
- **Best For**: Tech startups, modern products, Gen Z audience
- **Vibe**: Cool, modern, premium, Instagram-worthy

### 4️⃣ **Warm Orange** (`4-warm-orange.html`)
- **Background**: Warm cream-orange gradient
- **Accent Color**: Orange (#f97316 → #fb923c)
- **Style**: Friendly, approachable, energetic
- **Best For**: Community platforms, friendly products
- **Vibe**: Warm, welcoming, energetic, approachable

### 5️⃣ **Minimal Teal** (`5-minimal-teal.html`)
- **Background**: Light teal gradient with dot grid pattern (30px dots)
- **Accent Color**: Teal (#0d9488)
- **Style**: Clean, minimal, distraction-free
- **Best For**: Developer-focused tools, coding platforms
- **Vibe**: Clean, minimal, technical, focused
